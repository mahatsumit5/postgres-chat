Welcome to chat application backend api

To run this project

Clone this repo

git clone <url>

Install all the dependecies
npm install

To run this project
npm start

or npm run dev

ENVIRONMENT=Development
AWS_BUCKET_NAME=aws bucket name
AWS_REGION=Region
AWS_ACCESS_KEY=access_key
AWS_SECRET_KEY=Secret_key
